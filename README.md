# Embedded graphics coordinate transforms

[![Crates.io](https://img.shields.io/crates/v/embedded-graphics-coordinate-transform.svg)](https://crates.io/crates/embedded-graphics-coordinate-transform)
[![Docs.rs](https://docs.rs/embedded-graphics-coordinate-transform/badge.svg)](https://docs.rs/embedded-graphics-coordinate-transform)

Wrappers for rotating, mirroring, and transposing embedded graphics displays.

![A grid of displays with different coordinate transforms applied.](assets/overview.png)

## Minimum supported Rust version

The `embedded-graphics-coordinate-transform` crate is tested to compile on the latest stable Rust release.
